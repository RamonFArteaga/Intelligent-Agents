# Intelligent-Agents
This is my e-PortFolio for the Final Submission of Intelligent Agents Course at University of Essex. 

Student: Ramon Flores Arteaga
ePortFolio Website for Intelligent Agents: https://ramonfarteaga.github.io/Intelligent-Agents/

Repository URL: https://github.com/RamonFArteaga/Intelligent-Agents.git

IT Includes 9 tabs:

  ⊙ Home: Provides an overview of the module, the purpose of the e-Portfolio and direct access to the main project areas.
  
  ⊙ Reflection: Contains the final critical reflection, including discussion of 3 study cases such as: Bias in autonomous decision-making, Explainable decisions in agents (XAI),      Responsible   deployment of agents, including avoiding harmful or unintended emergent behaviours: Teamwork and professional development.
  
  ⊙ Project Proposal: Presents the Week 7 design of Inquiro, including the original architecture, agent roles, technology stack, requirements, risks and planned evaluation strategy.File       Proposal is the repository, Folder: Week 7 Project Proposal
  
  ⊙ Final Project: Describes the Week 11 implemented version of Inquiro, including the five-agent architecture, validation, testing, logging, error handling and final system behaviour. All    files are found in the repository, Folder: Week 11 Project Implementation.
  
  ⊙ Teamwork: Documents the collaborative planning process, team contributions, individual responsibilities and lessons learned from group work.
  
  ⊙ Discussions: Includes the initial posts, peer responses and summary posts completed during Units 1, 5 and 9.
  
  ⊙ Exercises: Contains practical activities from Units 6, 8 and 10, including agent communication using KQML and KIF, constituency-based parse trees and a deep learning application.
  
  ⊙ Critical Evaluation: Compares the Week 7 design with the Week 11 implementation and evaluates the project’s strengths, limitations, evidence, ethical implications and future     improvements.
  
  ⊙ About Me: Provides a brief professional and academic profile, including relevant experience, technical interests and career goals.
